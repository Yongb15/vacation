 switch-case문
	- 선택문
	- 같다 (==)

 switch {
	case 값1 :
		// 키워드 == 값1 실행되는 코드
	case 값2 :
		// 키워드 == 값1 실행되는 코드
	...
	default :
		// 키워드가 모든 값과 일치 하지 않을 때 실행되는 코드 // if문의 else와 같음.
 }
 
 		// 키워드 == 값1 -> 실행 시점, break를 만날 때 까지 실행		// break == 실행 시점을 멈추는 명령어

 값 : 정수

 JDK7 : 값 : 문자열 / 정수만 되는 특징이 바뀐건 없다!
 JDK8 : Enum 상수 / 정수만 되는 특징이 바뀐건 없다!

 반복문 
 
	- while문
		while(조건식) {
			// 조건식이 참일때 반복되는 코드
		}

		// 조건식이 참일 때만 실행

		// 반복문에서의 중요한 점
			- 초기값 : (int num = 1) 
			- 조건식 : (num <= 100)
			- 증감식 : (num++)

	- do-while문
		do {
			// 조건식이 참일때 반복되는 코드	
		}
		while(조건식);

		// 조건식이 참이 아니더라도 한번은 수행되는 코드

	- for문
		- 반복의 최적화된 구조 제공
		for (초기화식; 조건식; 증감식) {
			// 조건식이 참일 때 수행되는 코드 
		}

		순서 : index - 0부터 시작하는 순서 번호
		관례 : 초기화식 변수명으로 i를 사용하고, 중첩 for문에서는 알파벳 순서대로 i, j, k, ...

	- break
		- 반복 중단 명령어 (while, do-while, for)
	
	- continue 
		- 현재 반복을 중단하고 새로 시작 - 건너띄기

		tip) 짝수, 홀수 판별하는 방법 -> 2로 나누었을 때 나머지가 1이면 홀수

	- 중첩된 반복문
		- 반복문 안에 반복문이 있는 형태
		- while문 안에 while, do-while, for
		- for문 안에 for, while. do-while
		- for문 안에 for문이 있는 형태 : 중첩 for문

 배열
	- 변수 여러개를 한 번에 정의

 배열의 구조
	- 자료형이 같은 자료 여러개를 한 번에 관리
	- 자료가 연속으로 나열된 자료구조 (물리적으로 붙어있는 구조)

 배열 선언과 초기화
	- 변수의 일종, 변수 여러개를 한꺼번에 정의하는 방식
	
 배열 선언
	자료형[] 배열명 = new 자료형 [공간의 개수];
	- 자료형 배열명[] = new 자료형 [공간의 개수];	


 배열 초기화
	- 선언과 동시에 초기화

	자료형[] 배열명 = new 자료형 [] { 값1, 값2, 값3, ... };
	자료형[] 배열명 = { 값1, 값2, 값3, ... };

 인덱스 연산자
	- [] : 배열의 물리적 위치를 찾아주는 연산자, 0부터 시작
		ex) [0], [1], [...] 
	- 변수의 초기값
		- 정수 : 0 (byte, short, int, long)
		- 실수 : 0.0 (float, double)
		- 논리값(boolean) : false
		- 참조자료형 : null(값이 없음)
			ex) String
 length 속성
	- 공간의 개수
